# Developer Portfolio

A modern, responsive personal portfolio website built with HTML, CSS, and JavaScript. Designed to showcase projects, skills, and contact information through a clean UI, smooth animations, dark mode support, and a mobile-friendly layout.

## Features

* Responsive design for desktop and mobile
* Dark mode toggle
* Smooth scroll animations
* Project showcase section
* Skills and contact sections
* Clean and reusable design system

## Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Purpose

This project was created to demonstrate front-end development skills, UI design principles, and the ability to build a complete portfolio website from scratch.
